##Note:
AirFly Insights analyzes large-scale airline data to uncover operational trends and delay patterns. The project involves data cleaning, feature engineering, and calculating KPIs like On-Time Performance. Using Python, Matplotlib, and Seaborn, it provides visual insights into airport efficiency and route reliability to improve understanding of flight performance.
